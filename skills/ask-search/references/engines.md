# SearxNG 引擎列表

默认使用 Google，如需切换引擎请用 `-e` 指定。

```bash
ask-search "query"                    # 默认 google
ask-search "query" -e bing            # 切换到 bing
ask-search "query" -e arxiv,openalex  # 多引擎
```

## 通用搜索

google, bing, duckduckgo
